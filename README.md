# DS-2002 Final Project – Movie Data Lakehouse

## Overview
This project implements a Bronze–Silver–Gold data lakehouse architecture using PySpark.
It integrates data from MySQL, MongoDB Atlas, and CSV sources

## Data Sources
- Northwind_DW (MySQL) – Date Dimension
- Sakila (MySQL) – Film and Actor Data
- sample_mflix (MongoDB Atlas) – Movies, Users, Comments
- IMDb Top 1000 (CSV)
